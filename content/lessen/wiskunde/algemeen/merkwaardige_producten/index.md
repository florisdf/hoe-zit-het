## De familie van merkwaardige producten
$$(a + b ) \cdot (a + b) = a^2 + 2ab + b^2$$
$$(a - b ) \cdot (a - b) = a^2 - 2ab + b^2$$
$$(a + b ) \cdot (a - b) = a^2 - b^2$$

{{% attention "Te onthouden" %}}
De drie **merkwaardige producten** zijn:
\begin{equation}
    (a + b )^2 = a^2 + 2ab + b^2
\end{equation}
\begin{equation}
    (a - b )^2 = a^2 - 2ab + b^2
\end{equation}
\begin{equation}
    (a + b ) \cdot (a - b) = a^2 - b^2
\end{equation}
